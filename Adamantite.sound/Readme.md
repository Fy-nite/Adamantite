#Adamantite.Sound

Welcome to the official repository for Adamantite.Sound, a powerful and flexible audio library designed for game development and multimedia applications. This library provides a comprehensive set of tools for managing sound effects, music, and audio playback with ease.

the Sound Library works by manipulating PCM data directly, allowing for low-level control over audio playback. It supports various audio formats and provides features such as 3D sound positioning, volume control, and audio effects.
